The Yucca Packaging Website is a front-end web project that represents a packaging company's online presence. It includes multiple sections such as company information, product packaging services, and contact details.

This project demonstrates the ability to create a professional corporate website layout using pure HTML and CSS while maintaining responsive design and clean code structure.

🚀 Features:-
Responsive business website layout
Modern navigation bar
Product / packaging services sections
Informational company sections
Image-based product showcase
Clean footer with useful links
Mobile-friendly design

🛠️ Technologies Used:-
HTML5 – Website structure
CSS3 – Styling and layout design
Flexbox – Responsive alignment
CSS Flex – Structured section layout
